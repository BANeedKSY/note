# note株式会社 (東証:5243) プラットフォーム・ガバナンス監査アーカイブ

## 概要
本リポジトリは、note株式会社（以下、運営）が運営するプラットフォーム「note」において発生した、一利用者に対する不透明な言論封殺、およびガバナンス不全に関する客観的エビデンスを保全・公開するものです。

運営による「アカウント停止（BAN）」という強権発動により失われた情報を、改ざん不能な形式（PDFおよび魚拓）でアーカイブし、行政機関（総務省・消費者庁・警察）および資本提携先（株式会社KADOKAWA：東証9468）への報告資料として供します。

## 監査の背景
1. **医療情報の安全性に関する指摘:** 自称外科医による不適切な医療情報の拡散に対し、利用者の立場から修正勧告および通報を行ったものの、運営による不作為を確認。
2. **一方的な言論封殺:** 運営への公開質問状を「規約違反」として恣意的に非公開化。
3. **資本提携に伴うガバナンスの形骸化:** KADOKAWA社との提携発表と呼応するように、正当なパロディ（風刺）を「剽窃」と断じ、議論の場を物理的に遮断（BAN）。
4. **権限（sudo）の濫用:** 運営責任者がSNS上で「特権の行使」を示唆し、特定の利用者に対して心理的圧迫を加えた疑い。
